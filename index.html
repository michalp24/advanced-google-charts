Embedding Your Chart

Every chart exported from this tool is powered by Google Charts — a free, battle-tested visualization library with zero API keys required. Follow the steps below to embed any chart on any web page.

1

Load the Google Charts Library

Add this single <script> tag to the <head> of your HTML page. It loads the Charts loader from Google's CDN — no npm, no bundler, no API key needed.

<script type="text/javascript" src="https://www.gstatic.com/charts/loader.js"></script>
2

Place the Chart Container

Add a <div> with a unique id wherever the chart should appear. Set the desired width and height via CSS or inline styles.

<div id="chart_div" style="width:800px;height:460px;"></div>
3

Paste the Exported Code

Copy the embed code from the Export tab and paste it just before the closing </body> tag. It contains everything: your data, chart type, and all configuration options.

<!-- Paste the exported <script> block here --> </body>
4

Serve the Page Over HTTP(S)

Google Charts requires pages to be served over HTTP or HTTPS — opening an HTML file directly via file:// will not work. In development, use a local server:

# Python (built-in): python3 -m http.server 8080 # → visit http://localhost:8080 # Node.js (npx serve): npx serve . # Or any live-server, nginx, Apache, etc.
5

Embed via <iframe> (Optional)

If you downloaded the standalone HTML file, you can embed it in any other page using an <iframe>. This is useful for CMS platforms, Confluence, or any system that restricts inline scripts.

<iframe src="chart-embed.html" width="820" height="480" frameborder="0" style="border:none;"></iframe>
